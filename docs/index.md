# QuickLocalize — CSV → ScriptableObject

CSV를 읽어 **ScriptableObject** DB로 변환하는 Unity 에디터 툴 **QuickLocalize** 문서입니다.

## 시작하기
1) 패키지 임포트  
2) `Window > QuickLocalize > CSV Importer` 실행  
3) CSV 지정 → Import → `LocalizationDB.asset` 생성 확인
